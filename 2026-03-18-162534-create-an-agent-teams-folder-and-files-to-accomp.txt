  --dangerously-skip-permissions — this lets agents work autonomously without

   the same file, so parallel execution is safe.

  Why --print mode:
  - --print runs Claude Code non-interactively — it reads the prompt, executes,
  and outputs. Perfect for builder agents that have deterministic task
  specifications.
